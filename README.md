# 芬兰科研人员去哪儿？——城市迁移与学科变迁

基于 2008–2018 年芬兰科研人员流动数据，从城市空间与学科知识两个维度，以数据叙事的方式揭示学术人才的流动规律。

## 在线访问

[https://cdbc-258.github.io/finnish-researcher-mobility/](https://cdbc-258.github.io/finnish-researcher-mobility/)

## 页面结构

项目采用单页滚动叙事（scrollytelling）设计，包含以下四个场景：

### 01 流动有多普遍？

通过各城市留守率环图、留守率排行榜等图表，回答"科研人员到底流不流动"这个基本问题。

- **各城市留守率**：环形图展示同城留存 vs 跨城流出比例
- **留守率排行榜**：三个时间段并排对比，红→绿色阶表示低→高留守率

### 02 流动发生在哪里？

多维度分析城市间的迁移路径、网络结构和枢纽特征。

- **城市迁移网络**：力导向布局展示城市间迁移关系，边宽代表迁移强度
- **From-To 迁移矩阵**：对角线为同城留存，一目了然地对比城市间迁移动量
- **城市流入/流出排行**：柱状图直观展示各城市流动方向
- **同城留存 vs 跨城流动**、**净流入率**、**城市对方向性**
- **流出集中度（HHI）**：衡量流出目的地的集中程度
- **社区结构**：揭示城市迁移网络的社区划分
- 支持按时间段、方向筛选，城市搜索和路径数量控制

### 03 学科知识如何流动？

从学科大类角度观察知识在学科间的传递网络与结构，内置学科知识流动探索器，支持多种视图切换：

- **留守率环图**：各学科的自我引用比例
- **净流动**：学科间的净知识流向
- **热力矩阵**：学科 × 学科流动强度矩阵
- **学科网络**：学科间的关系网络图
- **弦图**：学科间流动的弦图可视化
- **开放度**：各学科的对外开放程度
- **差异热力图**：前后五年学科流动变化

支持动画开关、流速和密度调节。

### 04 时间变化与双视角对比

前后五年对比分析，城市流动与学科流动的异同比较。

- **城市流动时间变化**：趋势图展示流动格局的时变特征
- **城市差异热力图**：前后五年的流动变化差异
- **学科差异热力图**：学科间流动的时间变化
- **KPI 对比卡片**：城市流动 vs 学科流动的关键指标对比
- **共同特征与关键差异**：双视角总结洞察

## 功能特性

- **交互控制**：每个场景都提供时间段切换、方向筛选、搜索等控制
- **数据提示**：所有图表支持悬停查看详细数据
- **导航联动**：顶部导航栏随滚动高亮当前场景，支持点击跳转
- **响应式布局**：适配桌面和移动端

## 技术栈

- D3.js v7 — 全部图表渲染
- d3-sankey — 桑基图支持
- 纯静态 HTML / CSS / JS，无需构建工具
- 深色学术主题，Noto Serif SC + Inter 字体

## 数据来源

基于 2008–2018 年芬兰科研人员流动数据，涵盖城市间迁移矩阵与学科间知识流动网络。
